# 📖 hello_react_front-end <a name="about-project"></a>

**hello-react-front-end** is a demo that demonstrates the use of the RoR + React project as two apps, front-end and back-end. This repo represents the front-end part which is built with React and Redux. It displays a random message sent from the backend by making a simple get request.

- You can find the back-end app of this project using this [link](https://github.com/SelmaNdapanda/hello-rails-back-end/tree/back_end)

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- `node` installed in your computer, if not refer this [link](https://nodejs.dev/en/download/package-manager/) to install it in your system.

### Setup

Clone this repository to your desired folder:

```
  git clone https://github.com/SelmaNdapanda/hello-react-front-end.git

  cd hello-react-front-end

```

### Install

Run these commands to install the required dependencies:

```
  npm install

```

### Usage

To run the project, execute the following command:

```
  npm start
```
